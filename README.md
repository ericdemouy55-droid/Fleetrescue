# FleetPartner Rescue 24/7 — Prototype démo

Prototype Streamlit pour démontrer :
- demande chauffeur
- géolocalisation simulée
- recherche des dépanneurs les plus proches
- cascade de sollicitation
- acceptation / passage au suivant / clôture
- référentiel dépanneurs éditable
- reporting simple

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

Les CSV sont créés automatiquement dans le dossier `data` au premier lancement.
